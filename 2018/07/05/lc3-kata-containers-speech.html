Kata Containers Presentation at LC3 China 2018

Bergwolf TECHKata

LC3 China

It’s the second year of LC3 China. It is a big event for the Linux and Container groups in China. A lot of interesting and highly technical presentations, and a good marketing oppertunity of many companies.

Last year, we (hyper.sh) all attended but did not submit any presentations. This year, I presented “Kata Containers: When OCI Meets Virtulization”, Lai Jiangshan presented “Run Linux Kernel as a Daemon”, and Zhu Hui presented “The Implementation of Anti-Pages Fragmentation in Linux Kernel”.

Kata Containers: When OCI Meet Virtulization

The presentation first gives an introduction to the OCI (Open Container Initiative) spec and the vm-based container runtime – the kata containers project, focusing on its targeting threat model, high-level architecture and use cases. Then it dives deeper on how kata containers cope with limitations of the OCI (Open Container Initiative) spec that is the very basis of Docker containers, and how it manages to extend the spec for advanced features when integrating with kubernetes.
